# ESP32 - 42 BLACKHOLE VISUALISER

This little project will get the datas from 42's API and display the time remaining before student's blackhole

Is is made to be writen to a ESP32 with an oled screen (128 x 64), You can add buttons to navigate the list

## Settings

You'll have to change some values to match your campus and pool year
Also you must have the API sets to allow the esp to access the datas

Your credentials must be included within the code

Wifi have to be configured too
